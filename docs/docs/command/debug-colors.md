# miren debug colors

Print some colors

## Usage

```bash
miren debug colors [flags]
```

## Global Options

- `--options` — Path to file containing options
- `--server-address` — Server address to connect to (default: `127.0.0.1:8443`)
- `--verbose, -v` — Enable verbose output
