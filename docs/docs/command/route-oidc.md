# miren route oidc

OIDC authentication management for routes

## Usage

```bash
miren route oidc [flags]
```

## Subcommands

- [`miren route oidc disable`](/command/route-oidc-disable) — Disable OIDC authentication for a route
- [`miren route oidc enable`](/command/route-oidc-enable) — Enable OIDC authentication for a route
- [`miren route oidc show`](/command/route-oidc-show) — Show OIDC configuration for a route
